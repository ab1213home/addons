# Home Assistant 应用：openWakeWord

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

Home Assistant 应用（formerly known as add-on），使用 [pyopen-wakeword](https://github.com/rhasspy/pyopen-wakeword) 进行唤醒词检测。

这是 [Voice 年](https://www.home-assistant.io/blog/2022/12/20/year-of-voice/) 的一部分。

需要 Home Assistant 2023.9 或更高版本。
