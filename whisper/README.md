# Home Assistant 应用：Whisper

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

Home Assistant 应用（formerly known as add-on），使用多种语音转文字后端：

- [faster-whisper](https://github.com/guillaumekln/faster-whisper/)
- [HuggingFace transformers](https://huggingface.co/docs/transformers/index)
- [sherpa-onnx](https://k2-fsa.github.io/sherpa/onnx/index.html)（仅限 [parakeet](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v3)）
- [onnx-asr](https://github.com/istupakov/onnx-asr)（仅限 [GigaAM](https://github.com/salute-developers/GigaAM)）

这是 [Voice 年](https://www.home-assistant.io/blog/2022/12/20/year-of-voice/) 的一部分。
