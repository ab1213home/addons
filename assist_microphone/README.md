# Home Assistant 应用： Assist Microphone

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

Home Assistant 应用（formerly known as add-on），使用本地 USB 麦克风来控制 [Assist](https://www.home-assistant.io/voice_control/)。

这是 [Voice 年](https://www.home-assistant.io/blog/2022/12/20/year-of-voice/) 的一部分。
