# Home Assistant 应用：Letsencrypt

Let's Encrypt 是一个证书颁发机构，通过自动化流程提供免费的 X.509 证书用于传输层安全加密，消除了手动创建、验证、签名、安装和更新安全网站证书的复杂过程。

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

设置 Letsencrypt 允许您为您的网页和 web 界面使用经验证的证书。您需要拥有您申请证书的域名。

生成的证书可以在其他应用中使用。默认情况下，其他应用中证书的路径和文件将引用此应用内生成的证书。
