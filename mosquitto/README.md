# Home Assistant 应用：Mosquitto broker

Home Assistant 的 MQTT broker。

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

## 关于

您可以使用此应用（formerly known as add-on）安装 Eclipse Mosquitto，这是一个开源（EPL/EDL 许可）消息代理，实现了 MQTT 协议。Mosquitto 轻量级，适用于从低功耗单板计算机到完整服务器的所有设备。更多信息请参阅 [mosquitto]。
