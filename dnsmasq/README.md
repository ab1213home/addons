# Home Assistant 应用：Dnsmasq

一个简单的 DNS 服务器。

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

## 关于

设置和管理 Dnsmasq DNS 服务器。这允许您操作 DNS 请求。例如，您可以让您的 Home Assistant 域名使用网络内部的地址进行解析。
