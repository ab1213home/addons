# Home Assistant 应用：Samba 共享

使用 Windows 文件共享通过网络共享您的配置。

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

## 关于

此应用（formerly known as add-on）允许您通过网络在不同操作系统之间启用文件共享。它让您可以通过 Windows 和 macOS 设备访问您的配置文件。
