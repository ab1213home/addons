# Home Assistant 应用：OpenThread Border Router

OpenThread Border Router 应用（formerly known as add-on）。该应用使用上游 OpenThread Border Router 实现，并将其包装为 Home Assistant 的应用。

**注意：** 这需要一个支持 802.15.4 的无线电，并带有 OpenThread RCP 固件。如果您使用的是 [Home Assistant Yellow](https://www.home-assistant.io/yellow/) 或 [Home Assistant Connect ZBT-1](https://www.home-assistant.io/connectzbt1/)（之前称为 SkyConnect），则正确的固件会自动安装。

![支持 aarch64 架构][aarch64-shield]
![支持 amd64 架构][amd64-shield]

## 关于

此应用允许您创建或加入 Thread 网络，并将 Home Assistant 变为 Thread Border Router。
