# Home Assistant 应用：CEC Scanner

扫描并发现 HDMI CEC 设备及其地址。

![支持 aarch64 架构][aarch64-shield] ![支持 amd64 架构][amd64-shield]

此应用（formerly known as add-on）允许扫描 CEC 设备。它对于查找您设备的 CEC 地址非常有用。
